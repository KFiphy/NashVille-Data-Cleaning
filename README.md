# NashVille-Data-Cleaning

The Nashville housing dataset is uncleaned data consisting of information on house ownership in Nashville. 

# Task

My role as a data analyst is to clean up the dataset and get it ready for analysis and visualization. 

# Deliverables

This project demonstrates proficiency in SQL syntax such as SELECT, FROM, CONVERT, UPDATE, ALTER, DROP, ADD, SET, WHERE, ISNULL, COUNT, GROUP BY, INNER JOIN, SUBSTRING, CHARINDEX, LEN, PARSENAME, REPLACE, COUNT, CASE, ROWNUM, PARTITION BY, OVER, and CTE.
